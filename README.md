# React-Node-Websocket
Experimenting React + Node.js + [socket.io](https://socket.io/).  
It doesn't do much but it's enough to demonstrate the real-time power of using websocket.  
## Get started
* clone this repo
* cd client:
  * npm i
  * npm start
* cd server:
  * npm i
  * npm run dev
